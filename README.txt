# 🔧 Features

✅ Bridge Rectifier using 1N4007

✅ AC to DC Conversion

✅ LED Power Indicator

✅ Capacitor Filtering

✅ Ready for Manufacturing

---

# 🛠 Components Used

| Component | Value |
|------------|---------|
| D1-D4 | 1N4007 |
| C1 | 1000µF |
| R1 | 10K |
| R2 | 2.2K |
| D5 | LED |
| J1 | AC Input |
| J2 | DC Output |

---

# ⚙ Working Principle

1. AC voltage enters through J1.
2. Four 1N4007 diodes form a bridge rectifier.
3. AC is converted to pulsating DC.
4. Capacitor C1 smooths the output.
5. LED indicates power ON.
6. DC output is available at J2.

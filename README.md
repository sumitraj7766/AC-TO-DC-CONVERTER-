## 📖 Overview

This project presents the design and development of a compact **AC to DC Converter PCB** using **KiCad EDA**.

The circuit converts alternating current (AC) into filtered direct current (DC) through a bridge rectifier and smoothing capacitor network. The project was developed to gain practical experience in **schematic design, PCB layout, routing, DRC verification, Gerber generation, and PCB manufacturing workflows**.

---

## 🎯 Project Objectives

* Design a complete AC to DC conversion circuit
* Create a manufacturable PCB layout in KiCad
* Implement bridge rectification using 1N4007 diodes
* Reduce output ripple using capacitor filtering
* Provide visual power indication through an LED
* Generate industry-standard manufacturing files

---

## ⚙️ Working Principle

### Step 1: AC Input

Alternating current is applied through the AC input terminal (J1).

### Step 2: Rectification

Four 1N4007 diodes are configured as a bridge rectifier to convert AC into pulsating DC.

### Step 3: Filtering

A 1000µF capacitor smooths the pulsating DC and reduces ripple voltage.

### Step 4: Power Indication

An LED indicator confirms that the circuit is powered and operating.

### Step 5: DC Output

The filtered DC voltage is delivered through the output terminal (J2).
